## 🚀 Quick Start (Do This Now)

### Step 1: Pull the AI Model (5 minutes)
```powershell
# Open PowerShell and run:
ollama pull gemma2:2b
```
Or use the automated setup script:
```powershell
& D:\Hackk\CropIntel\setup-ollama.ps1
```

### Step 2: Start the Backend (30 seconds)
```powershell
cd D:\Hackk\CropIntel
python -m uvicorn price_api:app --reload --port 8000
```

You'll see startup output like:
```
[READY] Connected (gemma2:2b available)
[READY] Connected (1100 records)
[OK] Cache TTL: 3600s
[OK] RAG Engine: Ready
API endpoints ready at http://localhost:8000
```

### Step 3: Open Chatbot (10 seconds)
Open in your browser:
```
file:///d:/Hackk/CropIntel/chatbot.html
```

### Step 4: Test It
Ask a question:
- "What is the current price of wheat?"
- "Which crops are trending?"
- "Compare rice and maize prices"

## ⚠️ Troubleshooting

| Problem | Solution |
|---------|----------|
| "Model not found" | Run: `ollama pull gemma2:2b` |
| "Cannot connect" | Check Ollama is running or restart it |
| Slow first response | Normal - model is loading (10-15 seconds) |
| Out of memory | Close other apps, need 8GB+ RAM |
| Gemma2:2b not in list | Run pull command again |
